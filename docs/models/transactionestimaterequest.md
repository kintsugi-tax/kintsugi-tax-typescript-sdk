# TransactionEstimateRequest

Request model for tax estimation, including all fields from TransactionEstimateBase
and an additional field to simulate nexus being met.

## Example Usage

```typescript
import { TransactionEstimateRequest } from "@kintsugi-tax/tax-platform-sdk/models";

let value: TransactionEstimateRequest = {
  date: new Date("2024-10-17T04:28:55.337Z"),
  externalId: "<id>",
  currency: "CUC",
  addresses: [],
  transactionItems: [],
};
```
